# Smart Cached RAG Assistant

Полнофункциональная асинхронная RAG-система (Retrieval-Augmented Generation) общего назначения на базе OpenAI API,
векторного хранилища ChromaDB и встроенного SQLite-кеширования для оптимизации затрат и увеличения скорости ответов.
Проект также включает в себя модуль автоматической оценки качества ответов с помощью фреймворка Ragas.

## 🚀 Основные возможности

**Умное текстовое деление на чанки:** Собственный алгоритм разбиения текста на смысловые фрагменты
с сохранением контекста предложений (`overlap`) на стыках фрагментов.
**Векторный поиск:** Использование эмбеддингов `text-embedding-3-small` от OpenAI и легковесной векторной базы данных
ChromaDB с косинусным расстоянием для поиска релевантного контекста.
**Локальное SQLite кеширование:** Сохранение хэшированных поисковых запросов, ответов модели и использованного контекста
в файл `api_rag_cache.db`. При повторном обращении ответ отдается мгновенно без обращения к платному OpenAI API.
**Интерактивный CLI:** Удобный интерфейс командной строки с поддержкой просмотра статистики векторного хранилища/кеша
и возможностью его быстрой очистки.
**Автоматическая оценка (Ragas):** Встроенный скрипт для вычисления метрик `Faithfulness` (точность ответа по отношению
к контексту) и `Context Precision` (точность извлечения контекста).

## 🏗️ Архитектура проекта

Проект состоит из следующих ключевых модулей:

1. **`app.py`** — Главный скрипт запуска консольного интерфейса приложения, обработки команд пользователя и вывода
результатов/статистики.
2. **`rag_pipeline.py`** — Оркестратор RAG-системы. Связывает воедино проверку кеша, извлечение документов, формирование
системного промпта и генерацию ответа через LLM. 
3. **`vector_store.py`** — Модуль для работы с ChromaDB: чтение исходных документов, генерация эмбеддингов, деление
на чанки и семантический поиск.
4. **`cache.py`** — Реализация хранилища кеша на базе SQLite с автоматической нормализацией текста и SHA-256 хэшированием
запросов.
5. **`evaluate_ragas.py`** — Модуль тестирования и оценки качества работы системы на тестовом наборе вопросов с выводом
итогового балла.
